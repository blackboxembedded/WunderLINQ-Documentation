Introduction

Premièrement, merci d'avoir fait l'achat de votre WunderLINQ !

Cette documentation a pour but de vous aider à configurer votre WunderLINQ et de décrire les fonctionnalités disponibles dans l'application WunderLINQ.

Exigences

Pour utiliser l'application WunderLINQ, vous aurez besoin des éléments suivants.

  1. Module WunderLINQ
  2. Application WunderLINQ installée
  3. Téléphone ou tablette

Pour commencer

Insérez votre appareil WunderLINQ dans votre support de navigation et allumez le contact. Une fois allumé le témoins lumineux sur votre WunderLINQ devrait passé de vert à bleu clignotant.

Associez votre appareil WunderLINQ.

Accédez aux paramètres Bluetooth de votre appareil mobile. Sélectionnez l'appareil WunderLINQ pour établir l'association.

Le WunderLINQ devrait maintenant apparaître comme Connecté dans vos paramètres Bluetooth et le voyant d'état du WunderLINQ devrait être bleu.

Installez l'application WunderLINQ

Configuration initiale de l'app WunderLINQ

Accorder des autorisations

Lors du premier lancement de l'application WunderLINQ, l'assistant de démarrage vous sera présenté. L'assistant vous guide à travers toutes les autorisations dont l'application WunderLINQ a besoin pour fonctionner. Une fois celles-ci configurées, vous n'aurez plus l'assistant de démarrage à chaque fois. Le refus de ces autorisations entraînera une réduction ou la désactivation de fonctionnalités de l'application WunderLINQ. Pour ajuster les autorisations ultérieurement, vous devez utiliser les paramètres.

Avertissement d'utilisation

Lors du premier lancement de la journée, un message d'avertissement vous sera présenté. Vous devez accepter les termes avant l'utilisation de l'application.

Une fois la configuration initiale terminée, votre WunderLINQ devrait se reconnecter automatiquement à votre appareil lorsqu'il est allumé. Si vous avez une moto prise en charge, certaines données devraient être remplies. Veuillez noter que certaines valeurs telles que la pression des pneus n'affichent pas de lecture initiale tant que la moto n'a pas parcouru une certaine distance.

Utilisation de l'application WunderLINQ

Écran de navigation générale

Vous pouvez naviguer dans les différents écrans de l'application WunderLINQ en utilisant une des trois méthodes suivantes.

  1. Appuyez gauche et droite sur la molette de sélection
  2. Gestes de balayage vers la gauche et vers la droite sur l'écran
  3. Boutons de l'application

Écran de la moto

L'écran affiche une grille personnalisable de points de données fournis par votre moto et votre smartphone. L'état de la connexion Bluetooth, l'état de défaut actif et les boutons de quelques autres fonctions sont également disponibles. Pour personnaliser la grille, vous pouvez appuyer longuement sur une cellule ou via les paramètres de l'application.

Écran d'informations sur la moto

La page d'informations sur le vélo affiche le VIN de votre moto, la prochaine date d'entretien et la distance jusqu'au prochain entretien. Sous les informations de votre moto, il y a une option pour réinitialiser certaines des données de votre moto. Actuellement, vous pouvez réinitialiser les compteurs journaliers, de consommation de carburant et de vitesse moyenne.

Ecran voyages ; points de cheminements

L'écran Voyages et Waypoints vous permet de choisir parmi vos journaux de trajet et vos Waypoints enregistrés.

Affichage des enregistrement de trajets

L'écran trajets enregistrés, affiche les journaux de trajet enregistrés à l'aide de la tâche rapide Démarrer/Arrêter l'enregistrement du trajet. Appuyez sur le trajet que vous souhaitez afficher.

Ecran des trajets

L'écran Trajet affiche une carte du trajet et un résumé des données enregistrées et une étiquette modifiable. Touchez l'étiquette pour faire apparaître le clavier.

Menu contextuel qui comprend des options pour :

Partager Envoie le journal de trajet au format CSV par le biais de différentes méthodes.
Partager GPX Envoie le journal de trajet au format GPX par le biais de différentes méthodes.
Supprimer Supprime définitivement le journal des trajets.

Écran Waypoints (points de cheminement) enregistrés

L'écran "Saved Waypoints" affiche les points de cheminement enregistrés à l'aide de la tâche rapide "Save Waypoint". Touchez le waypoint que vous souhaitez gérer. Le bouton + ouvrira l'écran Ajouter un point de cheminement "waypoint".

Sauver le "Waypoint" à l'écran

L'écran "Saved Waypoint" affiche une carte avec le point de cheminement "waypoint", la date d'enregistrement, l'emplacement et une étiquette modifiable. Touchez l'étiquette pour faire apparaître le clavier.

Étiquette Champ pour attribuer une étiquette au "waypoint".

Menu contextuel qui comprend des options pour :

Partager Envoie un lien Google Maps par le biais de différentes méthodes.
Partager GPX Envoie un "Waypoint GPX" par le biais de différentes méthodes.
Naviguer Vous dirige vers le "Waypoint" dans votre application cartographique préférée.
Ouvrir dans l'application Map Ouvre le "Waypoint" dans votre application cartographique préférée.
Supprimer Supprime définitivement le "Waypoint" enregistré.

Ajouter un "Waypoint" à l'écran

L'écran "Ajouter un waypoint" affiche une carte avec une épingle indiquant l'emplacement actuellement sélectionné. Pour ajouter un nouveau "Waypoint", utilisez la recherche d'adresse ou entrez une latitude/longitude ou déposez manuellement l'épingle sur la carte en touchant l'emplacement sur la carte. Lorsque vous êtes prêt, appuyez sur "Enregistrer"

Écran Hardware matériel

L'écran paramètres "hardware" permet d'accéder à tous les paramètres matériels WunderLINQ. Actuellement, ces paramètres contrôlent les performances de la roue de sélection et du port USB.

L'utilisation du bouton en bas de cet écran vous permettra de passer en mode Clé personnalisée. Le mode personnalisé ouvre la possibilité de personnaliser les touches du clavier envoyées par les différentes entrées Moto ainsi que de contrôler le port d'alimentation USB.

HW_SETTINGS_ACTION_MODE_SCREEN_DETAILS

HW_SETTINGS_ACTION_MODE_DEFAULT_TITLE HW_SETTINGS_ACTION_MODE_DEFAULT_DETAILS
HW_SETTINGS_ACTION_MODE_CUSTOM_TITLE HW_SETTINGS_ACTION_MODE_CUSTOM_DETAILS

Le bouton Reset réinitialisera tous les paramètres personnalisés aux valeurs d'usine par défaut.

En mode personnalisé, vous pouvez configurer les paramètres individuels. Le bouton en bas de cet écran vous permettra de revenir au mode par défaut.

L'écran de contrôle du port USB est utilisé pour régler l'état du port d'alimentation USB.

Les écrans de sensibilité sont utilisés pour régler la sensibilité de la pression longue ou de la double pression (RT/K1600).

L'écran Key mapping est lancé en sélectionnant l'action que vous souhaitez modifier une fois en mode personnalisé. Cette fonctionnalité est destinée aux utilisateurs avancés qui comprennent l'utilisation des mappages dans Android et iOS.

Après avoir enregistré vos paramètres, veuillez retirer votre WunderLINQ de la station d'accueil, puis réinsérez-le pour effectuer une réinitialisation complète. N'oubliez pas que la modification du mappage des touches peut avoir un impact sur votre capacité à naviguer dans l'application WunderLINQ avec la molette.

Mise à jour du micrologiciel

L'application WunderLINQ-DFU ou WunderLINQ Device Firmware Update est une application distincte utilisée pour mettre à jour le logiciel qui s'exécute réellement sur le matériel WunderLINQ. Les mises à jour du micrologiciel ne doivent être appliquées que si vous avez besoin de la nouvelle fonctionnalité ou du correctif de bogue.

Accédez à l'App Store d'Apple et installez l'application.

Il est très important de suivre les étapes telles qu'elles sont écrites, en particulier en ce qui concerne les étapes d'appariement/dissociation.

Dissocier le WunderLINQ

Pour commencer, nous allons faire en sorte que votre appareil mobile oublie l'appairage WunderLINQ. L'écran de lancement initial vous rappellera que vous devrez oublier l'appairage WunderLINQ et vous fournira un bouton pour ouvrir vos paramètres Bluetooth.

Une fois dans vos paramètres Bluetooth, recherchez votre WunderLINQ sous Mes appareils. Appuyez sur l'icône bleue à côté du nom de l'appareil WunderLINQ.

Sélectionnez Oublier. Après avoir supprimé le jumelage, vous devez redémarrer votre appareil mobile.

Mettre à jour le micrologiciel WunderLINQ

Une fois que vous avez oublié le couplage entre votre WunderLINQ et votre appareil mobile, vous êtes prêt à mettre à jour votre WunderLINQ. Commencez par allumer le WunderLINQ en mettant le contact de votre moto, la LED WunderLINQ doit clignoter en bleu.

N'essayez pas de mettre à jour votre WunderLINQ pendant que la moto est en marche ou si votre moto ou votre appareil mobile a une batterie faible.

Lancez l'application WunderLINQ-DFU. Si vous avez supprimé le jumelage, sélectionnez OK pour continuer.

Votre appareil WunderLINQ doit être répertorié. S'il n'est pas répertorié même après quelques rafraîchissements, essayez de remettre le WunderLINQ son support.

Lorsque vous sélectionnez le WunderLINQ, iOS vous demandera de le coupler au WunderLINQ. À ce stade, vous souhaitez sélectionner Annuler.

Une fois connecté au WunderLINQ, nous devons le mettre en mode DFU pour accepter une mise à jour. Appuyez sur le bouton Activer le mode DFU en bas de l'écran. Une fois sélectionné, le WunderLINQ redémarrera en mode DFU, cela peut être confirmé par une LED verte fixe. L'application reviendra à l'écran précédent de la liste des appareils.

Votre appareil WunderLINQ doit être répertorié. Essayez de rafraîchir si nécessaire. Sélectionnez WunderLINQ.

Lorsque vous sélectionnez le WunderLINQ, iOS vous demandera de vous coupler au WunderLINQ. À ce stade, vous souhaitez sélectionner Pair.

Après un couplage et une connexion réussis, l'écran de mise à niveau du micrologiciel s'affiche. Si vous utilisez le firmware >= 1.7, vous verrez la version actuellement installée. Appuyez sur Sélectionner le micrologiciel.

Sélectionnez la version du micrologiciel que vous souhaitez installer et appuyez sur MISE À NIVEAU

Le firmware va maintenant être téléchargé sur le WunderLINQ.

Une fois le téléchargement terminé, WunderLINQ redémarrera en mode normal. Vous devrez maintenant suivre les étapes pour dissocier à nouveau le WunderLINQ avant de pouvoir réparer et utiliser l'application WunderLINQ.

Dépannage

Aucune donnée de performances

Si aucune donnée de performance n'est observée dans l'application, vérifiez d'abord que vous disposez d'une moto prise en charge. Si votre vélo moto manque de données de performance, il affichera généralement le VIN partiel sous la page "Informations sur la moto" de l'application WunderLINQ. Si la moto doit avoir des données de performance, une séquence de réinitialisation du réseau de l'appareil doit être suivie.

Contrôle partiel via la roue

Si certaines commandes via la molette ne s'enregistrent pas. Tout d'abord, assurez-vous que vous utilisez le mappage de touches par défaut sous Configuration Hardware. Si vous exécutez le mappage de touches par défaut, essayez une réinitialisation. Si le problème persiste après avoir suivi ces étapes, il est possible que votre roue soit défaillante ou partiellement défaillante. Nous avons observé un petit nombre de cas de dysfonctionnement de la roue.

Pas de contrôle avec la molette

Si votre moto est équipée d'un combiné d'instruments TFT, vous devez le mettre en mode Nav ou menu Nav sur le RT/K1600, pour transmettre les appuis sur les roues au berceau. Vérifiez ensuite que vous utilisez le mappage de touches par défaut sous le paramètre Hardware. Si vous exécutez le mappage de touches par défaut, essayez une réinitialisation de l'appareil.

Comment effectuer une réinitialisation de votre appareil mobile

En dernier recours, une réinitialisation de votre appareil mobile peut être effectuée. Nous avons vu cela résoudre un certain nombre de problèmes de connexion. Veuillez noter que vous perdrez tous vos couplages Bluetooth et WiFi précédents après avoir effectué ces étapes.

  1. Oubliez le couplage WunderLINQ sous les paramètres Bluetooth iOS
  2. Réinitialiser les paramètres réseau sous les paramètres de réinitialisation iOS
  3. Redémarrez l'appareil iOS
  4. Associez le WunderLINQ sous les paramètres Bluetooth iOS
  5. Lancez l'application WunderLINQ et testez

Rapport de problème

Si vous pensez avoir trouvé un bogue ou un défaut, vous pouvez nous envoyer un rapport en utilisant le bouton Envoyer les journaux sur l'écran À propos. Si votre problème est lié aux données de performance du vélo ou au contrôle des roues, il peut être utile d'activer la journalisation de débogage dans les paramètres de l'application tout en reproduisant votre problème, puis en envoyant les journaux. Si vous êtes à l'aise avec GitHub, vous pouvez vérifier si le bogue a déjà été enregistré ou soumettre votre bogue directement là-bas.
